# CX Interactive Experience Lab

**Estúdio premium de criação de experiências interativas em HTML.**

Este repositório é uma base profissional para desenvolver conteúdos digitais de alto impacto — materiais que combinam storytelling executivo, design cinematográfico e interatividade com função cognitiva.

Cada HTML produzido aqui deve parecer uma mistura de keynote premium, playbook executivo, manifesto visual e experiência de aprendizagem interativa.

---

## Propósito

Criar experiências digitais de pensamento sobre:
- Customer Experience e Customer Success
- Inteligência Artificial e dados
- Liderança e gestão contemporânea
- Tendências e futuro dos negócios

Não produzimos páginas. Produzimos materiais de referência.

---

## Estrutura do Repositório

```
├── CLAUDE.md              # Diretrizes e padrões para IA assistente
├── prompts/               # Prompts mestres para geração de conteúdo
├── skills/                # Modos especialistas para diferentes capacidades
├── references/            # Checklists e materiais de referência
├── templates/             # Templates base reutilizáveis
└── output/                # HTMLs gerados (produção final)
```

### Pastas

| Pasta | Papel |
|-------|-------|
| `prompts/` | Prompts mestres e estruturados para geração de HTMLs premium |
| `skills/` | Perfis especialistas que definem como pensar em cada dimensão do projeto |
| `references/` | Checklists de qualidade, guias de estilo e materiais de apoio |
| `templates/` | Templates HTML base, elegantes e reutilizáveis |
| `output/` | Resultado final — os HTMLs produzidos pelo estúdio |

---

## Fluxo de Trabalho

```
1. Definir tese central
2. Definir headline e promessa intelectual
3. Estruturar narrativa (arco + seções)
4. Definir proposta visual (paleta, tipografia, ritmo)
5. Definir proposta interativa (quais interações e por quê)
6. Gerar o HTML
7. Refinar design, UX, copy e ritmo visual
8. Validar com checklist final (/references/quality-check.md)
```

---

## Princípios

1. **Nunca genérico** — cada material traz um olhar novo
2. **Storytelling executivo** — narrativa sofisticada, não decorativa
3. **Interatividade cognitiva** — cada interação ensina algo
4. **Presente + Futuro** — repertório atual com perspectiva de futuro
5. **Impacto visual** — estética editorial e cinematográfica
6. **Clareza com densidade** — profundidade sem complicação
7. **Forma e conteúdo unidos** — design, narrativa e interação como sistema

---

## Tecnologias

- HTML5 semântico
- CSS moderno (Custom Properties, Grid, Flexbox, animações)
- JavaScript vanilla (ou módulos leves quando necessário)
- Vite (para desenvolvimento quando estruturado)
- Single-file HTML como formato principal de entrega

---

## Como Usar

1. Consulte `CLAUDE.md` para diretrizes completas
2. Use os prompts em `/prompts` como ponto de partida
3. Ative os skills relevantes em `/skills` conforme o projeto
4. Gere o HTML seguindo o fluxo de trabalho
5. Valide com `/references/quality-check.md`
6. Exporte para `/output`
